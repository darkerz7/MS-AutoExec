# MS-AutoExec for ModSharp
Automatically executes commands after events

## Required packages:
1. [ModSharp](https://github.com/Kxnrl/modsharp-public)
2. [AdminManager](https://github.com/Kxnrl/modsharp-public/tree/master/Sharp.Modules/AdminManager)

## Installation:
1. Compile or copy MS-AutoExec to `sharp/modules/MS-AutoExec` folger
2. Configure config file `config.json`

## Admin privileges
Privilege | Description
--- | ---
`autoexec:reload` | Allows reloading configs

## Commands:
`ms_ae_reload` - Reload config file of AutoExec(`autoexec:reload` permission)
